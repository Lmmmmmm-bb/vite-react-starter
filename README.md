## Getting started

1. Clone the template repo.

```bash
degit git@github.com:Lmmmmmm-bb/vite-react-starter.git my-project
```

2. Initialize git repository.

```bash
git init
```

3. Install dependencies & Start dev server.

```bash
pnpm i
pnpm dev
```
